# 金番写作 MVP - 界面说明文档索引

## 一、文档说明

本目录按“一个界面一个文件”的方式拆分前后台界面说明。

每个界面文档统一说明：

- 界面定位。
- 界面长什么样。
- 需要实现的功能。
- 交互说明。
- 状态说明。
- MVP 边界。

这些文档用于辅助前端开发、后端接口拆分、测试验收和后续原型设计。

## 四、开发使用建议

建议开发时按以下顺序阅读：

1. 先看 `前后台界面规划文档.md`，理解整体结构。
2. 再看本索引，定位具体界面文件。
3. 开发某个页面前，阅读对应界面说明文件。
4. 后续接口文档和任务拆分应以这些界面文件为基础继续细化。
