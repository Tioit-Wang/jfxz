# 用户端 - 作品工作台页界面说明

## 一、界面定位

作品工作台页是用户端核心界面，作者的大部分创作行为都在这里完成。

对应路由：`/books/:bookId`

## 二、界面长什么样

采用“上方工具栏 + 下方左中右三栏”的结构。

```text
┌──────────────────────────────────────────────┐
│ 顶部栏：作品名 / 保存状态 / 积分 / 作品信息 / 角色 / 设定 / 账户 │
├──────────────┬────────────────┬──────────────┤
│ 左侧章节列表 │ 中间正文编辑器 │ 右侧 AI 对话 │
│ 新建章节     │ 章节标题       │ 会话标题      │
│ 章节卡片     │ 正文输入区     │ 消息流        │
│ 提要预览     │ 字数统计       │ 输入框        │
└──────────────┴────────────────┴──────────────┘
```

## 三、需要实现的功能

- 加载当前作品信息。
- 展示当前作品名称。
- 加载章节列表。
- 切换当前章节。
- 编辑章节标题、正文、章节提要。
- 展示保存状态。
- 展示字数统计。
- 展示右侧 AI 对话面板。
- 打开作品信息、角色、设定、套餐、账户等大模态框。

## 四、交互说明

- 进入页面后默认加载最近编辑章节。
- 如果没有章节，左侧展示空状态并引导新建章节。
- 点击章节后，中间编辑器加载对应章节。
- 编辑正文后触发自动保存。
- 点击 AI 分析，对当前章节进行分析并生成高亮。
- 点击顶部“角色”，打开角色管理模态框。
- 点击顶部“设定”，打开设定管理模态框。
- 点击顶部“作品信息”，打开作品信息模态框。
- 未登录访问时弹出登录注册模态框。

## 五、状态说明

- 页面加载中。
- 作品不存在。
- 无权限访问。
- 章节为空。
- 保存中。
- 已保存。
- 保存失败。
- AI 分析中。

## 六、MVP 边界

- 不做多作品同屏。
- 不做多章节同时编辑。
- 不做富文本编辑器。
- 不做复杂自定义布局。
