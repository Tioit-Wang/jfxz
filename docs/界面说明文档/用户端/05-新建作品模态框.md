# 用户端 - 新建作品模态框界面说明

## 一、界面定位

新建作品模态框用于创建一本新的作品。创建成功后进入该作品工作台。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 新建作品              │
│ 作品名称              │
│ 短简介                │
│ 梗概                  │
│ 题材标签              │
│ 背景与世界规则          │
│ 重点要求，可选          │
│ 禁忌要求，可选          │
│ 取消 / 创建作品         │
└──────────────────────┘
```

## 三、需要实现的功能

- 填写作品基础信息。
- 填写作品总纲信息。
- 创建作品。
- 创建成功后跳转作品工作台。
- 必填项校验。

## 四、交互说明

- 点击作品列表页“新建作品”打开。
- 点击取消关闭模态框。
- 点击创建作品时校验必填字段。
- 创建成功后跳转 `/books/:bookId`。
- 创建失败展示错误信息。

## 五、状态说明

- 默认填写状态。
- 提交中。
- 校验失败。
- 创建失败。

## 六、MVP 边界

- 不做 AI 自动创建完整作品。
- 不做作品模板。
- 不做多版本总纲。
