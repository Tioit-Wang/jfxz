# 用户端 - 作品信息模态框界面说明

## 一、界面定位

作品信息模态框用于查看和编辑当前作品的基础信息和总纲信息。

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 作品信息                      │
├──────────────────────────────┤
│ 作品名称                      │
│ 短简介                        │
│ 梗概                          │
│ 题材标签                      │
│ 背景与世界规则                  │
│ 重点要求                       │
│ 禁忌要求                       │
├──────────────────────────────┤
│ 取消 / 保存                    │
└──────────────────────────────┘
```

## 三、需要实现的功能

- 查看当前作品完整字段。
- 编辑作品名称。
- 编辑短简介。
- 编辑梗概。
- 编辑题材标签。
- 编辑背景与世界规则。
- 编辑重点要求。
- 编辑禁忌要求。
- 保存修改。

## 四、交互说明

- 从作品工作台顶部“作品信息”打开。
- 点击保存后更新作品信息。
- 保存成功后关闭或保留在当前弹窗，按产品体验决定。
- 保存后 AI 对话和编辑器分析使用最新作品上下文。
- 点击取消时，如果有未保存内容，需要二次确认。

## 五、状态说明

- 加载中。
- 编辑中。
- 保存中。
- 已保存。
- 保存失败。

## 六、MVP 边界

- 不做版本历史。
- 不做多总纲。
- 不做故事线模块。
