# 用户端 - 章节列表区域界面说明

## 一、界面定位

章节列表区域位于作品工作台左侧，用于管理当前作品下的章节结构。

## 二、界面长什么样

```text
┌──────────────┐
│ 章节          │
│ + 新建章节    │
├──────────────┤
│ 第 1 章 标题  │
│ 提要预览      │
├──────────────┤
│ 第 2 章 标题  │
│ 提要预览      │
└──────────────┘
```

## 三、需要实现的功能

- 展示章节列表。
- 展示章节顺序。
- 展示章节标题。
- 展示章节提要预览。
- 新建章节。
- 删除章节。
- 调整章节顺序。
- 切换当前章节。

## 四、交互说明

- 点击章节卡片：中间编辑器加载对应章节。
- 点击新建章节：创建占位章节并进入编辑状态。
- 点击删除：弹出二次确认。
- 删除当前章节后自动切换到相邻章节或空状态。
- 调整顺序后刷新章节编号。

## 五、状态说明

- 章节加载中。
- 无章节空状态。
- 当前章节高亮。
- 删除中。
- 排序保存中。

## 六、MVP 边界

- 不做批量导入导出。
- 不做章节分卷。
- 不做章节版本历史。
