# 用户端 - @ 引用选择浮窗界面说明

## 一、界面定位

@ 引用选择浮窗用于在 AI 对话输入框中快速引用章节、角色和设定。

## 二、界面长什么样

```text
┌──────────────┐
│ 最近选择      │
│ 角色：张三    │
│ 设定：京城    │
├──────────────┤
│ 章节：第 3 章 │
│ 章节：第 2 章 │
└──────────────┘
```

## 三、需要实现的功能

- 输入 `@` 后打开浮窗。
- 无关键字时展示最近选择和章节列表。
- 有关键字时搜索章节、角色、设定。
- 键盘上下选择。
- 回车确认。
- Esc 关闭。
- 选中后插入结构化 `@名称` Mention 节点。
- 维护最近选择。

## 四、交互说明

- 输入 `@`：打开浮窗。
- 输入关键字：实时过滤结果。
- 按上下键：移动高亮项。
- 按回车：选中当前项。
- 按 Esc：关闭浮窗。
- 点击浮窗外：关闭浮窗。
- 选中后关闭浮窗，并在输入框插入引用节点。
- 引用节点作为一个整体参与编辑，Backspace/Delete 一次删除整个节点。
- 发送时引用节点会转成结构化 references，不只依赖输入框中的普通文本。

## 五、状态说明

- 无关键字状态。
- 搜索中。
- 无搜索结果。
- 最近选择为空。

## 六、MVP 边界

- 不做跨作品引用。
- 不做复杂高级搜索。
- 不做多选引用面板。
