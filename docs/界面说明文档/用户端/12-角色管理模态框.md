# 用户端 - 角色管理模态框界面说明

## 一、界面定位

角色管理模态框用于管理当前作品中的人物资料。

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 角色管理                      │
├──────────────┬───────────────┤
│ 搜索框        │ 角色详情        │
│ 角色列表      │ 名称            │
│ 新建角色      │ 简介            │
│              │ 详情            │
│              │ 编辑 / 删除 / 复制 │
└──────────────┴───────────────┘
```

## 三、需要实现的功能

- 角色列表。
- 按名称和简介搜索。
- 新建角色。
- 查看角色详情。
- 编辑角色。
- 删除角色。
- 快速复制角色名称、简介、详情。

## 四、交互说明

- 点击顶部栏“角色”打开模态框。
- 点击角色列表项，右侧显示详情。
- 点击新建角色，打开角色表单。
- 点击编辑，进入编辑状态或打开编辑表单。
- 点击删除，弹出二次确认。
- 保存成功后刷新列表和详情。

## 五、状态说明

- 加载中。
- 空角色列表。
- 搜索无结果。
- 保存中。
- 删除中。
- 保存失败。

## 六、MVP 边界

- 不做人物关系图。
- 不做角色版本历史。
- 不做角色自动去重。
