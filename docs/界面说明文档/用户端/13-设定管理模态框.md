# 用户端 - 设定管理模态框界面说明

## 一、界面定位

设定管理模态框用于管理当前作品中的非人物设定，例如地点、装备、规则、组织等。

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 设定管理                      │
├──────────────┬───────────────┤
│ 类型筛选      │ 设定详情        │
│ 搜索框        │ 类型            │
│ 设定列表      │ 名称            │
│ 新建设定      │ 简介            │
│              │ 详情            │
│              │ 编辑 / 删除 / 复制 │
└──────────────┴───────────────┘
```

## 三、需要实现的功能

- 设定列表。
- 按设定名称搜索。
- 按设定类型筛选。
- 新建设定。
- 查看设定详情。
- 编辑设定。
- 删除设定。
- 快速复制设定名称、简介、详情。

## 四、交互说明

- 点击顶部栏“设定”打开模态框。
- 选择类型后刷新列表。
- 输入关键字后实时搜索。
- 点击设定项后右侧展示详情。
- 点击新建设定打开表单。
- 点击删除需要二次确认。
- 保存成功后刷新列表和详情。

## 五、状态说明

- 加载中。
- 空设定列表。
- 搜索无结果。
- 保存中。
- 删除中。
- 保存失败。

## 六、MVP 边界

- 不做复杂关系图。
- 不做设定版本历史。
- 不做自动抽取和自动去重。
