# 用户端 - 会话历史模态框界面说明

## 一、界面定位

会话历史模态框用于查看和切换当前作品下的 AI 会话。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 会话历史              │
│ + 新建会话            │
├──────────────────────┤
│ 会话标题              │
│ 最近消息预览 / 时间    │
├──────────────────────┤
│ 会话标题              │
│ 最近消息预览 / 时间    │
└──────────────────────┘
```

## 三、需要实现的功能

- 展示最近会话列表。
- 展示会话标题。
- 展示最近消息预览。
- 展示最近活跃时间。
- 新建会话。
- 切换会话。

## 四、交互说明

- 从右侧 AI 对话面板点击“历史”打开。
- 点击某个会话，关闭模态框并切换当前会话。
- 点击新建会话，创建新的会话线程并切换过去。
- 会话按最近活跃时间倒序排列。

## 五、状态说明

- 加载中。
- 无历史会话。
- 切换中。
- 创建失败。

## 六、MVP 边界

- 不做会话搜索。
- 不做会话重命名。
- 不做置顶、归档、删除。
