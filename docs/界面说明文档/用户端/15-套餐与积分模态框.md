# 用户端 - 套餐与积分模态框界面说明

## 一、界面定位

套餐与积分模态框用于展示用户当前积分余额、当前套餐，以及可购买的套餐和加油包。

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 套餐与积分                    │
├──────────────────────────────┤
│ 当前套餐 / 月度积分 / 加油包积分 │
├──────────────────────────────┤
│ 套餐列表                      │
│ 套餐名 / 价格 / 月度积分 / 附带加油包 │
├──────────────────────────────┤
│ 加油包列表                    │
│ 名称 / 价格 / 积分数量 / 有效期 │
└──────────────────────────────┘
```

## 三、需要实现的功能

- 展示当前套餐。
- 展示月度积分。
- 展示加油包积分。
- 展示可购买套餐。
- 展示可购买加油包。
- 创建套餐订单。
- 创建加油包订单。
- 打开支付二维码模态框。

## 四、交互说明

- 点击顶部栏积分或套餐入口打开。
- 点击购买套餐，创建订单并打开支付二维码。
- 点击购买加油包，创建订单并打开支付二维码。
- 支付成功后刷新积分余额和当前套餐。

## 五、状态说明

- 加载中。
- 无可购买套餐。
- 无可购买加油包。
- 创建订单中。
- 创建订单失败。
- 支付成功后刷新中。

## 六、MVP 边界

- 不做优惠券。
- 不做折扣码。
- 不做发票。
- 不做积分转赠。
