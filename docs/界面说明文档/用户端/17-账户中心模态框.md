# 用户端 - 账户中心模态框界面说明

## 一、界面定位

账户中心模态框用于展示和修改当前用户的基础账户信息。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 账户中心              │
├──────────────────────┤
│ 邮箱                  │
│ 昵称                  │
│ 账户状态              │
│ 当前套餐              │
│ 当前积分              │
├──────────────────────┤
│ 保存昵称 / 退出登录    │
└──────────────────────┘
```

## 三、需要实现的功能

- 展示邮箱。
- 展示昵称。
- 修改昵称。
- 展示账户状态。
- 展示当前套餐。
- 展示当前积分。
- 退出登录。

## 四、交互说明

- 点击顶部栏头像或账户入口打开。
- 修改昵称后点击保存。
- 保存成功后顶部栏同步更新昵称。
- 点击退出登录后清理登录态，并回到落地页或作品列表未登录状态。

## 五、状态说明

- 加载中。
- 保存中。
- 保存失败。
- 退出确认。

## 六、MVP 边界

- 不做头像上传。
- 不做密码修改。
- 不做邮箱验证。
- 不做账号注销。
