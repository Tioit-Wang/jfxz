# 管理端 - 管理员登录页界面说明

## 一、界面定位

管理员登录页用于管理员进入后台管理系统。管理端需要独立入口，避免和普通用户端混淆。

对应路由：`/admin/login`

## 二、界面长什么样

```text
┌──────────────────────┐
│ 金番写作管理端          │
├──────────────────────┤
│ 邮箱                  │
│ 密码                  │
│ 登录按钮               │
└──────────────────────┘
```

## 三、需要实现的功能

- 管理员邮箱密码登录。
- 校验账号角色是否为 admin。
- 登录成功进入管理端。
- 登录失败展示错误。
- 被禁用账号禁止登录。

## 四、交互说明

- 输入邮箱和密码。
- 点击登录。
- 系统校验账号密码。
- 系统校验角色是否为管理员。
- 登录成功跳转 `/admin`。
- 登录失败展示提示。

## 五、状态说明

- 默认输入状态。
- 登录中。
- 账号密码错误。
- 非管理员账号。
- 账号被禁用。

## 六、MVP 边界

- 不做验证码。
- 不做找回密码。
- 不做多管理员权限分级。
