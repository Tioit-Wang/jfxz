# 管理端 - 后台基础布局界面说明

## 一、界面定位

后台基础布局是所有管理端页面共用的外壳，包括顶部栏、左侧菜单和主内容区。

对应路由：`/admin/*`

## 二、界面长什么样

```text
┌──────────────────────────────────────────────┐
│ 顶部栏：金番写作管理端 / 当前管理员 / 退出登录 │
├──────────────┬───────────────────────────────┤
│ 左侧菜单      │ 主内容区                       │
│ 用户管理      │                               │
│ 商品管理      │                               │
│ 订单管理      │                               │
│ 订阅管理      │                               │
│ 会话管理      │                               │
│ 系统配置      │                               │
└──────────────┴───────────────────────────────┘
```

## 三、需要实现的功能

- 展示管理端系统名。
- 展示当前管理员信息。
- 左侧菜单导航。
- 当前菜单高亮。
- 退出登录。
- 主内容区渲染具体管理页面。

## 四、交互说明

- 点击左侧菜单切换管理模块。
- 点击退出登录，清理管理端登录态并返回 `/admin/login`。
- 非管理员访问时跳转登录页或展示无权限。

## 五、状态说明

- 管理员信息加载中。
- 无权限。
- 菜单切换中。

## 六、MVP 边界

- 不做复杂仪表盘。
- 不做菜单权限配置。
- 不做多主题切换。
