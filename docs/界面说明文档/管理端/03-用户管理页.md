# 管理端 - 用户管理页界面说明

## 一、界面定位

用户管理页用于管理员查看用户列表、搜索用户、查看用户详情和控制用户状态。

对应路由：`/admin/users`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 用户管理                      │
│ 搜索框：邮箱/昵称              │
├──────────────────────────────┤
│ 表格：邮箱 / 昵称 / 角色 / 状态 / 注册时间 / 最近登录 / 操作 │
├──────────────────────────────┤
│ 分页                          │
└──────────────────────────────┘
```

用户详情可以用右侧抽屉或大模态框展示。

## 三、需要实现的功能

- 用户列表。
- 按邮箱搜索。
- 按昵称搜索。
- 查看用户角色。
- 查看用户状态。
- 查看注册时间。
- 查看最近登录时间。
- 查看用户详情。
- 启用用户。
- 禁用用户。

## 四、交互说明

- 输入关键字后点击搜索或回车搜索。
- 点击查看详情，打开用户详情抽屉。
- 点击禁用，弹出二次确认。
- 点击启用，弹出二次确认或直接启用。
- 状态变更成功后刷新当前行。

## 五、状态说明

- 加载中。
- 搜索中。
- 无搜索结果。
- 状态修改中。
- 状态修改失败。

## 六、MVP 边界

- 不做用户标签。
- 不做用户分组。
- 不做批量导入导出。
- 不做复杂权限配置。
