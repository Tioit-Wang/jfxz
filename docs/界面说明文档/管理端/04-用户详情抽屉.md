# 管理端 - 用户详情抽屉界面说明

## 一、界面定位

用户详情抽屉用于查看某个用户的基础信息、当前订阅和积分余额。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 用户详情              │
├──────────────────────┤
│ 邮箱                  │
│ 昵称                  │
│ 角色                  │
│ 状态                  │
│ 注册时间              │
│ 最近登录时间           │
├──────────────────────┤
│ 当前订阅              │
│ 月度积分              │
│ 加油包积分             │
└──────────────────────┘
```

## 三、需要实现的功能

- 查看用户基础信息。
- 查看账户状态。
- 查看当前订阅状态。
- 查看积分余额。
- 从详情中启用或禁用用户。

## 四、交互说明

- 从用户管理页点击“详情”打开。
- 点击关闭返回用户列表。
- 点击禁用或启用后刷新详情和列表。

## 五、状态说明

- 详情加载中。
- 加载失败。
- 状态修改中。

## 六、MVP 边界

- 不做编辑用户资料。
- 不做手动调整积分。
- 不做手动修改订阅。
