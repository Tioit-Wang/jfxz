# 管理端 - 套餐与加油包管理页界面说明

## 一、界面定位

套餐与加油包管理页用于管理员维护用户端可购买的套餐和加油包商品。

对应路由：`/admin/products`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 商品管理                      │
│ Tab：套餐 / 加油包             │
├──────────────────────────────┤
│ 套餐表格或加油包表格            │
│ 新建按钮                       │
└──────────────────────────────┘
```

## 三、需要实现的功能

套餐 Tab：

- 套餐列表。
- 新建套餐。
- 编辑套餐。
- 启用套餐。
- 停用套餐。

加油包 Tab：

- 加油包列表。
- 新建加油包。
- 编辑加油包。
- 启用加油包。
- 停用加油包。

## 四、交互说明

- 点击 Tab 切换套餐和加油包。
- 点击新建打开商品表单弹窗。
- 点击编辑打开商品表单弹窗。
- 点击启用/停用，弹出确认。
- 保存成功后刷新列表。

## 五、状态说明

- 列表加载中。
- 空列表。
- 保存中。
- 启停中。
- 操作失败。

## 六、MVP 边界

- 不做优惠券。
- 不做限时活动。
- 不做分渠道定价。
- 不做复杂促销组合。
