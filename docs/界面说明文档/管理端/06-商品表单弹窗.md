# 管理端 - 商品表单弹窗界面说明

## 一、界面定位

商品表单弹窗用于新建或编辑套餐、加油包。

## 二、界面长什么样

套餐表单：

```text
┌──────────────────────┐
│ 新建/编辑套餐          │
│ 套餐名称               │
│ 价格                  │
│ 月度积分               │
│ 附带加油包积分          │
│ 状态                  │
│ 取消 / 保存            │
└──────────────────────┘
```

加油包表单：

```text
┌──────────────────────┐
│ 新建/编辑加油包        │
│ 加油包名称             │
│ 价格                  │
│ 积分数量               │
│ 有效期                 │
│ 状态                  │
│ 取消 / 保存            │
└──────────────────────┘
```

## 三、需要实现的功能

- 新建套餐。
- 编辑套餐。
- 新建加油包。
- 编辑加油包。
- 表单校验。
- 保存商品。

## 四、交互说明

- 从商品管理页点击新建或编辑打开。
- 点击保存前校验必填字段。
- 保存成功后关闭弹窗并刷新列表。
- 点击取消关闭弹窗。
- 有未保存变更时关闭需要确认。

## 五、状态说明

- 默认编辑状态。
- 校验失败。
- 保存中。
- 保存失败。

## 六、MVP 边界

- 不做商品图片。
- 不做复杂富文本说明。
- 不做销售渠道配置。
