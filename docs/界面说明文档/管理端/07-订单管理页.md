# 管理端 - 订单管理页界面说明

## 一、界面定位

订单管理页用于查看用户购买套餐或加油包产生的订单。

对应路由：`/admin/orders`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 订单管理                      │
│ 搜索/筛选区域                  │
├──────────────────────────────┤
│ 表格：订单编号 / 用户 / 商品类型 / 商品名称 / 金额 / 状态 / 支付时间 / 操作 │
├──────────────────────────────┤
│ 分页                          │
└──────────────────────────────┘
```

## 三、需要实现的功能

- 订单列表。
- 查看订单编号。
- 查看用户。
- 查看商品类型。
- 查看商品名称。
- 查看金额。
- 查看订单状态。
- 查看支付时间。
- 查看订单详情。
- 查看支付记录。

## 四、交互说明

- 输入条件后搜索订单。
- 点击详情打开订单详情抽屉。
- 点击支付记录打开支付记录详情。
- 列表按创建时间或支付时间倒序展示。

## 五、状态说明

- 加载中。
- 搜索中。
- 无订单。
- 无搜索结果。
- 详情加载失败。

## 六、MVP 边界

- 不做后台手动收款。
- 不做手动补单。
- 不做退款处理。
- 不做财务结算报表。
