# 管理端 - 订阅管理页界面说明

## 一、界面定位

订阅管理页用于查看用户当前套餐订阅状态。

对应路由：`/admin/subscriptions`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 订阅管理                      │
│ 搜索/筛选区域                  │
├──────────────────────────────┤
│ 表格：用户 / 套餐 / 状态 / 开始时间 / 结束时间 / 下次续费 / 操作 │
├──────────────────────────────┤
│ 分页                          │
└──────────────────────────────┘
```

## 三、需要实现的功能

- 订阅列表。
- 查看用户。
- 查看当前订阅套餐。
- 查看订阅状态。
- 查看开始时间。
- 查看结束时间。
- 查看下次续费时间。
- 查看订阅详情。
- 查看关联订单。

## 四、交互说明

- 搜索用户或套餐。
- 点击详情打开订阅详情抽屉。
- 点击关联订单跳转或打开订单详情。

## 五、状态说明

- 加载中。
- 无订阅。
- 搜索无结果。
- 详情加载失败。

## 六、MVP 边界

- 不做手动修改订阅。
- 不做手动续费。
- 不做取消订阅操作。
