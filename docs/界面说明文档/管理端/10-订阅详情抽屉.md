# 管理端 - 订阅详情抽屉界面说明

## 一、界面定位

订阅详情抽屉用于查看单个订阅的完整基础信息和关联订单。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 订阅详情              │
├──────────────────────┤
│ 用户                  │
│ 套餐                  │
│ 状态                  │
│ 开始时间              │
│ 结束时间              │
│ 下次续费时间           │
├──────────────────────┤
│ 关联订单列表           │
└──────────────────────┘
```

## 三、需要实现的功能

- 查看订阅基础信息。
- 查看关联订单。
- 从关联订单查看订单详情。

## 四、交互说明

- 从订阅管理页点击详情打开。
- 点击关联订单可跳转订单管理页或打开订单详情抽屉。
- 点击关闭返回订阅列表。

## 五、状态说明

- 加载中。
- 加载失败。
- 无关联订单。

## 六、MVP 边界

- 不做订阅编辑。
- 不做手动补发权益。
- 不做订阅状态人工流转。
