# 管理端 - 会话管理页界面说明

## 一、界面定位

会话管理页用于管理员只读查看用户 AI 会话，用于排查问题。

对应路由：`/admin/sessions`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 会话管理                      │
│ 搜索：用户 / 作品 / 会话标题    │
├──────────────────────────────┤
│ 表格：用户 / 作品 / 会话标题 / 来源 / 最近活跃时间 / 操作 │
├──────────────────────────────┤
│ 分页                          │
└──────────────────────────────┘
```

## 三、需要实现的功能

- 会话列表。
- 按用户搜索。
- 按作品搜索。
- 按会话标题搜索。
- 查看会话详情。
- 只读查看历史消息。

## 四、交互说明

- 输入条件后搜索会话。
- 点击详情打开会话详情抽屉。
- 点击历史记录查看消息列表。
- 消息内容只读，不允许编辑。

## 五、状态说明

- 加载中。
- 无会话。
- 搜索无结果。
- 详情加载失败。

## 六、MVP 边界

- 不做手动编辑会话内容。
- 不做批量删除会话。
- 不做会话分析报表。
- 不做跨会话检索。
