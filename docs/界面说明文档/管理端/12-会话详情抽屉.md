# 管理端 - 会话详情抽屉界面说明

## 一、界面定位

会话详情抽屉用于查看某个 AI 会话的索引信息、Agent Session 信息和历史消息。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 会话详情              │
├──────────────────────┤
│ 用户                  │
│ 作品                  │
│ 会话标题              │
│ 来源                  │
│ 最近活跃时间           │
│ Agent Session 信息     │
├──────────────────────┤
│ 历史消息，只读          │
└──────────────────────┘
```

## 三、需要实现的功能

- 查看会话基础信息。
- 查看 Agent Session 基础信息。
- 查看历史消息。
- 按时间顺序展示消息。

## 四、交互说明

- 从会话管理页点击详情打开。
- 历史消息区域可以滚动。
- 消息只读，不能编辑或删除。
- 点击关闭返回列表。

## 五、状态说明

- 加载中。
- 消息加载中。
- 无历史消息。
- 加载失败。

## 六、MVP 边界

- 不做消息编辑。
- 不做消息删除。
- 不做会话导出。
