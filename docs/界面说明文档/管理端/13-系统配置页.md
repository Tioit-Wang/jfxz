# 管理端 - 系统配置页界面说明

## 一、界面定位

系统配置页用于管理员维护系统级配置，MVP 阶段重点是支付宝当面付配置。

对应路由：`/admin/configs`

## 二、界面长什么样

```text
┌──────────────────────────────┐
│ 系统配置                      │
├──────────────┬───────────────┤
│ 配置分组      │ 配置项列表       │
│ payment.xxx  │ key / type / value / required / 操作 │
└──────────────┴───────────────┘
```

## 三、需要实现的功能

- 配置分组列表。
- 配置项列表。
- 展示配置键。
- 展示配置类型。
- 展示当前值。
- 展示是否必填。
- 编辑配置。
- secret 类型字段掩码展示。

## 四、交互说明

- 点击配置分组，右侧显示该分组下配置项。
- 点击编辑，打开配置编辑弹窗。
- secret 字段默认隐藏。
- 点击显示按钮后临时展示 secret 值。
- 保存成功后刷新配置列表。

## 五、状态说明

- 加载中。
- 无配置项。
- 保存中。
- 保存失败。
- secret 已隐藏。

## 六、MVP 边界

- 不做配置版本回滚。
- 不做配置审批流。
- 不做多环境配置。
- 不做配置变更审计报表。
