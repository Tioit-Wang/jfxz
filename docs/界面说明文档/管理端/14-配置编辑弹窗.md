# 管理端 - 配置编辑弹窗界面说明

## 一、界面定位

配置编辑弹窗用于编辑单个系统配置项。

## 二、界面长什么样

```text
┌──────────────────────┐
│ 编辑配置              │
├──────────────────────┤
│ 配置键，只读           │
│ 配置类型，只读          │
│ 是否必填，只读          │
│ 配置值输入框            │
│ 取消 / 保存            │
└──────────────────────┘
```

## 三、需要实现的功能

- 查看配置键。
- 查看配置类型。
- 查看是否必填。
- 按配置类型编辑配置值。
- 保存配置。

## 四、交互说明

- 从系统配置页点击编辑打开。
- 根据配置类型展示不同输入控件。
- secret 类型使用密码输入框。
- 点击保存前校验必填项。
- 保存成功后关闭弹窗并刷新配置列表。

## 五、状态说明

- 默认编辑状态。
- 校验失败。
- 保存中。
- 保存失败。

## 六、MVP 边界

- 不做配置预发布。
- 不做配置历史对比。
- 不做批量编辑。
