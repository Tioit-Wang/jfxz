# 妙蛙写作 MVP - 会话管理需求文档

## 一、目标

管理端会话模块用于查看用户会话索引和 Agent 会话基础信息，便于排查问题。

## 二、功能范围

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **AS1** | 会话列表       | 展示所属用户、所属作品、会话标题、来源、最近活跃时间。               |
| **AS2** | 会话搜索       | 支持按用户、作品、会话标题搜索。                                     |
| **AS3** | 会话详情查看   | 查看会话索引信息和 Agno Session 基础信息。                           |
| **AS4** | 历史记录查看   | 支持只读方式查看会话历史，用于定位问题。                             |
| **AS5** | 会话筛选       | 支持按用户、作品、会话来源和最近活跃时间范围筛选。                   |

筛选项：

- 用户。
- 作品。
- 会话来源：`manual`、`editor`。
- 最近活跃时间范围。

## 三、MVP 不做

- 手动编辑会话内容
- 批量删除会话
- 会话分析报表
- 跨会话检索
