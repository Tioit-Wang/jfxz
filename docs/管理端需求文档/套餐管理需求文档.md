# 金番写作 MVP - 套餐与积分权益包管理需求文档 V3.0

## 一、目标

管理端套餐管理模块用于维护 VIP 套餐和积分权益包商品，并支持查看和调整用户积分。

## 二、功能范围

### 2.1 套餐管理

| 编号 | 功能 | 描述 |
| :--- | :--- | :--- |
| **A1** | 套餐列表 | 展示所有套餐：名称、价格、每日 VIP 积分、附带积分权益包积分、套餐时长、状态。 |
| **A2** | 新建套餐 | 支持创建套餐商品，配置：名称、价格、每日 VIP 积分、附带积分权益包积分、套餐时长、状态。 |
| **A3** | 编辑套餐 | 支持修改套餐配置。修改仅影响新购买和新续费订单，已购买权益不变。 |
| **A4** | 套餐启停 | 支持启用或停用套餐。停用后用户端不再展示，已购买和已续费权益不受影响。 |
| **A4-1** | 套餐筛选 | 支持按状态、价格范围、每日 VIP 积分范围和套餐时长筛选套餐。 |

### 2.2 积分权益包管理

| 编号 | 功能 | 描述 |
| :--- | :--- | :--- |
| **A5** | 积分权益包列表 | 展示所有积分权益包商品：名称、价格、积分数量、状态。 |
| **A6** | 新建积分权益包 | 支持创建积分权益包商品，配置：名称、价格、积分数量、状态。 |
| **A7** | 编辑积分权益包 | 支持修改积分权益包配置。修改仅影响新购买订单，已购买权益不变。 |
| **A8** | 积分权益包启停 | 支持启用或停用积分权益包。停用后用户端不再展示，已购买权益不受影响。 |
| **A8-1** | 积分权益包筛选 | 支持按状态、价格范围和积分数量范围筛选积分权益包。 |

### 2.3 用户积分管理

| 编号 | 功能 | 描述 |
| :--- | :--- | :--- |
| **A9** | 用户积分查看 | 查看任意用户的 VIP 套餐状态、生效/到期时间、续费周期、VIP 每日积分余额、积分权益包余额和积分流水。 |
| **A10** | 手动调整积分 | 管理员手动给用户补发或扣减积分，需选择调整 VIP 每日积分或积分权益包余额，记录操作原因并生成对应流水。 |

筛选项：

- 套餐管理：状态、价格范围、每日 VIP 积分范围、套餐时长。
- 积分权益包管理：状态、价格范围、积分数量范围。
- 用户积分查看：用户、VIP 套餐状态、总积分余额范围。

## 三、积分字段规则

- 套餐的每日 VIP 积分和附带积分权益包积分保留 2 位小数。
- 积分权益包积分数量保留 2 位小数。
- 商品价格由管理员直接配置。管理端编辑商品时，系统根据当前积分汇率（`billing.points_per_cny`）显示价格与积分的换算参考。
- 积分权益包永久有效，不配置有效期。

## 四、关键业务规则

- **权益锁定**：用户已购买的套餐和积分权益包权益在下单支付时锁定，管理员后续修改商品配置不影响已购买权益。
- **续费顺延**：用户在 VIP 未到期时继续购买套餐，新套餐从当前 VIP 到期时间后顺延生效。
- **不可升级降级**：MVP 不支持套餐升级、降级、差价补购或立即切换权益。
- **停用不退款**：商品停用后仅影响新购买，已购买权益继续按购买时快照生效。
- **手动调整留痕**：管理员手动调整积分必须记录操作原因，生成对应积分流水。

## 五、MVP 不做

- 优惠券
- 限时活动
- 分渠道定价
- 复杂促销组合
- 自动退款
- 批量操作（如批量发放积分）
- 积分权益包到期规则
- 套餐升级、降级或差价切换
