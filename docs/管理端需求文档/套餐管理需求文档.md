# 金番写作 MVP - 套餐管理需求文档

## 一、目标

管理端套餐管理模块用于维护套餐和加油包商品。

## 二、功能范围

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **AP1** | 套餐列表       | 展示套餐名称、价格、月度积分、附带加油包积分、状态。                 |
| **AP2** | 新建套餐       | 支持创建套餐商品。                                                   |
| **AP3** | 编辑套餐       | 支持修改套餐价格、积分额度、状态。                                   |
| **AP4** | 加油包列表     | 展示加油包名称、价格、积分数量、有效期、状态。                       |
| **AP5** | 新建加油包     | 支持创建加油包商品。                                                 |
| **AP6** | 编辑加油包     | 支持修改加油包价格、积分数量、有效期、状态。                         |
| **AP7** | 商品启停       | 支持启用或停用套餐和加油包。                                         |

## 三、积分字段规则

- 套餐的月度积分和附带加油包积分保留 2 位小数。
- 加油包积分数量保留 2 位小数。
- 商品价格仍由管理员直接配置，不根据积分面值自动反推。

## 四、MVP 不做

- 优惠券
- 限时活动
- 分渠道定价
- 复杂促销组合
