# 金番写作 MVP - 套餐管理需求文档 V2.0

## 一、目标

管理端套餐管理模块用于维护套餐和字数包商品，并支持查看和调整用户积分。

## 二、功能范围

### 2.1 套餐管理

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **A1** | 套餐列表       | 展示所有套餐：名称、价格、每日积分数、附带字数包积分数、状态。       |
| **A2** | 新建套餐       | 支持创建套餐商品，配置：名称、价格、每日发放积分数、附带字数包积分数、状态。 |
| **A3** | 编辑套餐       | 支持修改套餐配置（价格、每日积分数、字数包积分数、状态等）。修改仅影响新购买用户，已购买用户权益不变。 |
| **A4** | 套餐启停       | 支持启用或停用套餐。停用后用户端不再展示，已购买的不受影响。         |

### 2.2 字数包管理

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **A5** | 字数包列表     | 展示所有字数包商品：名称、价格、积分数量、有效期天数、状态。         |
| **A6** | 新建字数包     | 支持创建字数包商品，配置：名称、价格、积分数量、有效期天数、状态。   |
| **A7** | 编辑字数包     | 支持修改字数包配置（价格、积分数、有效期、状态等）。同样新购买生效。 |
| **A8** | 字数包启停     | 支持启用或停用字数包。停用后用户端不再展示，已购买的不受影响。       |

### 2.3 用户积分管理

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **A9** | 用户积分查看   | 查看任意用户的积分余额、VIP套餐状态（生效/到期时间）、字数包批次列表、积分流水。 |
| **A10**| 手动调整积分   | 管理员手动给用户补发或扣减积分，需记录操作原因，生成对应流水。       |

## 三、积分字段规则

- 套餐的每日积分数和附带字数包积分数保留 2 位小数。
- 字数包积分数量保留 2 位小数。
- 商品价格由管理员直接配置，不根据积分面值自动反推。

## 四、关键业务规则

- **配置锁定**：用户已购买的套餐/字数包权益在购买时锁定，管理员后续修改商品配置不影响已购买用户的权益。
- **停用不退款**：商品停用后仅影响新购买，已购买的权益正常到期。
- **手动调整留痕**：管理员手动调整积分必须记录操作原因，生成对应积分流水。

## 五、MVP 不做

- 优惠券
- 限时活动
- 分渠道定价
- 复杂促销组合
- 自动退款
- 批量操作（如批量发放积分）
