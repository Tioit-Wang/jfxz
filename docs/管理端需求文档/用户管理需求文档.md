# 金番写作 MVP - 用户管理需求文档

## 一、目标

管理端用户管理模块用于查看用户、检索用户，并对用户账号状态做最小控制。

## 二、功能范围

| 编号   | 功能         | 描述                                                         |
| :----- | :----------- | :----------------------------------------------------------- |
| **AU1** | 用户列表     | 展示用户邮箱、昵称、角色、状态、注册时间、最近登录时间。     |
| **AU2** | 用户搜索     | 支持按邮箱、昵称搜索用户。                                   |
| **AU3** | 用户筛选     | 支持按角色、账号状态、注册时间范围和最近登录时间范围筛选。   |
| **AU4** | 账户状态管理 | 支持把用户设置为 `active` 或 `disabled`。                    |
| **AU5** | 用户详情查看 | 支持查看用户的基础信息、当前订阅状态和积分余额。积分余额保留 2 位小数。 |

筛选项：

- 角色：`user`、`admin`。
- 账号状态：`active`、`disabled`。
- 注册时间范围。
- 最近登录时间范围。

## 三、MVP 不做

- 用户标签体系
- 用户分组
- 复杂权限配置
- 批量导入导出
