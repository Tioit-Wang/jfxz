# 金番写作 MVP - 系统配置需求文档

## 一、目标

管理端系统配置模块用于维护全局配置项，支持不同类型字段，并为支付等基础能力提供统一配置入口。

## 二、功能范围

| 编号   | 功能             | 描述                                                                 |
| :----- | :--------------- | :------------------------------------------------------------------- |
| **AC1** | 配置列表         | 展示配置分组、配置键、类型、当前值、是否必填。                       |
| **AC2** | 配置编辑         | 支持按字段类型编辑配置值。                                           |
| **AC3** | 分组查看         | 支持按配置分组查看，例如 `payment.alipay_f2f`。                      |
| **AC4** | 密文字段展示     | `secret` 类型字段默认掩码显示。                                      |
| **AC5** | 配置初始化       | 系统初始提供默认 seed 配置项，值默认为空。                           |

## 三、支付宝当面付配置项

MVP 阶段管理端需要支持以下配置项：

- `payment.alipay_f2f.enabled`
- `payment.alipay_f2f.app_id`
- `payment.alipay_f2f.app_private_key`
- `payment.alipay_f2f.alipay_public_key`
- `payment.alipay_f2f.notify_url`
- `payment.alipay_f2f.seller_id`
- `payment.alipay_f2f.timeout_express`

## 四、MVP 不做

- 配置版本回滚
- 配置发布审批流
- 多环境配置管理
- 配置变更审计报表
