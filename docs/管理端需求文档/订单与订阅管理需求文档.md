# 金番写作 MVP - 订单与订阅管理需求文档

## 一、目标

管理端订单与订阅模块用于查看购买记录和当前订阅情况。

## 二、功能范围

| 编号   | 功能           | 描述                                                                 |
| :----- | :------------- | :------------------------------------------------------------------- |
| **AO1** | 订单列表       | 展示订单编号、用户、商品类型、商品名称、金额、状态、支付时间。       |
| **AO2** | 订单详情       | 查看订单基础信息和积分发放来源。                                     |
| **AO3** | 订阅列表       | 展示用户当前订阅套餐、状态、开始时间、结束时间、下次续费时间。       |
| **AO4** | 订阅详情       | 查看某条订阅的基础信息和关联订单。                                   |
| **AO5** | 支付记录查看   | 查看订单关联的支付记录、支付渠道状态、二维码创建情况和通知结果。     |

详细支付方案见 `../支付接入需求文档.md`。

## 三、MVP 不做

- 后台手动收款
- 手动补单流程
- 财务结算报表
- 自动退款处理
